# Smart Traffic Signal Control System

## Overview

This project implements an adaptive traffic signal control system capable of dynamically adjusting signal timings based on real-time traffic density.

Unlike conventional fixed-time traffic lights, the system monitors vehicle presence using sensors and allocates green signal durations according to traffic conditions, improving traffic flow efficiency and reducing unnecessary waiting times.

---

## Problem Statement

Traditional traffic signals operate using predefined timing sequences regardless of actual traffic conditions. This often leads to increased congestion, longer waiting times, and inefficient utilization of road infrastructure.

The objective of this project is to develop a smart traffic control system that responds to real-time traffic density and optimizes signal allocation accordingly.
